# 🎂 Online Cake Shop Website

A responsive and modern **Online Cake Shop Website** built using **HTML, CSS, and JavaScript**.  
This project showcases a beautiful UI for a bakery where users can explore products and contact the shop.

---

## 🚀 Features
- 🍰 Attractive homepage with banner
- 🛍️ Product section (cakes, donuts, etc.)
- 📱 Fully responsive design (mobile-friendly)
- 🍔 Hamburger menu for small screens
- 📩 Contact form
- 🎨 Clean and modern UI

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
